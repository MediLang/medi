Medi v0.0.3 Released

Date: 2025-09-30

Added

  • LLVM backend integration (feature llvm-backend) via medic_codegen_llvm (Inkwell/LLVM 15).
  • CI artifacts for targets: x86-64, wasm32-wasi, and initial riscv32.
  • CI matrix: fmt/lint, tests (Linux/macOS/Windows), quick bench smoke, per-target codegen artifacts.

Improved

  • Clinician-friendly diagnostics in parser (e.g., = vs ==, unmatched brackets, per suggestions).
  • Unified lexer error tokens across streaming and chunked paths; better parser recovery and EOF handling.
  • Clippy/formatting fixes; pre-commit hooks green.

Examples

  • Minimal codegen smoke programs emitted per target in CI for inspection.

Notes

  • Requires system LLVM 15.x when enabling llvm-backend.
