Medi v0.0.4 Released

Date: 2025-10-24

Highlights

  • Deterministic scheduler tests: Ack/barrier-based tests; serialized to avoid cross-talk.
  • Unified runtime diagnostics: RuntimeError, MemoryErrorKind, SchedulerErrorKind, RuntimeDiagnostic with context via medi_runtime_report!.
  • RT memory observability: region overflow and pool edge-case tags.
  • Channels & GC tags: std and crossbeam channel ops; gc.lock on poison.
  • Docs: new medic_runtime README covering errors, recovery, RT memory, and examples.

Performance

  • Diagnostics validated under ~1ms/event; RT latency tests remain green.
